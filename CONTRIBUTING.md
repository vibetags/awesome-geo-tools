# Contributing to Awesome GEO Tools

Thank you for your interest in contributing!

## Guidelines

### Adding a Tool or Resource

1. **Fork** this repository
2. **Add** your tool/resource in the appropriate category
3. **Follow** the existing format (table rows for tools, bullet points for resources)
4. **Submit** a pull request with a clear description

### Requirements for Tool Submissions
- Must be directly related to GEO (Generative Engine Optimization) or AI search visibility
- Must have a working website or public repository
- Must include: Name, Description, Type, and Link
- Commercial and open-source tools are both welcome

### Requirements for Resource Submissions
- Must be relevant to GEO practitioners
- Must be publicly accessible
- Must include a working link

### What We Don't Accept
- Duplicate entries
- Spam or self-promotion without value
- Tools/resources unrelated to GEO
- Broken links

## Categories

If your tool/resource doesn't fit an existing category, propose a new one in your PR description.

## Code of Conduct

Be respectful, inclusive, and constructive. We're building a community resource for the GEO ecosystem.
